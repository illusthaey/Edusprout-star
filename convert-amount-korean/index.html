금액 한글 변환기

숫자를 한 칸에 줄바꿈으로 붙여넣으면, 회계 개요 작성용 문구로 한 번에 바꿔줍니다. 엑셀 열/메모장 숫자 목록 복사 붙여넣기용입니다.

출력 형식
표기 기준
괄호 안 한글 앞에도 금을 붙여서 출력합니다. 예: 금2,200원(금이천이백원)
미리보기 금100,000원(금일십만원)

입력

Ctrl + Enter로 바로 변환

숫자는 줄바꿈 기준으로 구분합니다. 쉼표, 공백, 원 표시(원, ₩, ￦)는 자동 정리합니다.

결과

숫자를 입력하면 바로 변환됩니다.

빈 줄은 그대로 유지합니다. 숫자가 아닌 줄은 [입력 오류]로 표시합니다.

처리 규칙 보기
  • 일 생략 없음: 10 = 일십, 100 = 일백, 1,000 = 일천, 10,000 = 일만원처럼 처리합니다.
  • 기본 모드는 회계 개요 작성용 형식인 금1,140원(금일천일백사십원) 입니다.
  • 한글 금액만 모드로 바꾸면 금일천일백사십원처럼 출력합니다.
  • 입력 예: 100000, 100,000, ₩100,000, 100,000원 모두 처리합니다.
  • 0은 영원으로 처리합니다.

단일 HTML 파일이라 인터넷 없이도 실행할 수 있습니다.
