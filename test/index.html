특근매식비 업무 간소화 도구

·나이스 복무에서 공무원 초과근무 내역, 공무직 연장근로내역 CSV 형식 파일로 저장하고 업로드 하면 됨.

① 공무원 초과근무 미입력

나이스 CSV/XLSX 파일
예: 공무원_초과근무목록.csv
열 설정 (기본: 헤더1행 / 날짜B / 이름E·괄호제거)

② 공무직 연장근로 미입력

📂 나이스 CSV/XLSX 파일
예: 공무직_연장근로목록.csv
열 설정 (기본: 헤더1행 / 날짜B / 이름F)

③ 지문 증빙 내역 미입력

📂 지문인식기 XLSX/CSV
문막초 기본: B=발생시각, D=출퇴근, F=성명
열 설정 (학교별 업체 서식 다를 수 있음)

※ 날짜열에 "2026-04-24 09:07:24"처럼 시간이 붙어있어도 자동으로 날짜만 추출

④ 네이버폼 설문 응답 미입력

📂 네이버폼 XLSX
설문 질문 순서에 따라 열 위치 조정
열 설정 (기본: 헤더1행 / 이름D / 날짜F)
