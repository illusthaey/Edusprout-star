계약제교원 퇴직금 산정기

나이스에서 연도별 급여 총 지급현황을 엑셀 파일(.xslx)로 다운받아서 올리면 퇴직금 내역 산정해줌. 결과 파일 저장 가능

  • 기본 산식은 1일 평균임금*30일*(재직일수/365)입니다.
  • 평균임금이 통상임금보다 적으면 통상임금 기준으로 최종 지급액을 선택합니다.
  • 평균임금의 3개월은 90일 고정이 아니라 달력상 3개월입니다.
  • 가족수당·자녀학비보조수당·보육수당 등 복리후생성 항목은 평균임금과 통상임금에서 제외합니다.
  • 성과상여금은 지급월만으로 귀속 근무기간을 판별하기 어려워 파일에서 자동 반영하지 않고, 해당 계약기간 근무분만 수기 입력으로 반영합니다.
  • 원단위 절삭함.

1. 파일 업로드

여러 연도 파일을 한 번에 올릴 수 있습니다.

파일을 여기로 끌어다 놓거나 클릭해서 선택하세요
.xlsx / .xls · 여러 연도 파일 동시 업로드 가능
선택된 파일 없음
개인정보 보호를 위해 업로드 파일명은 화면에 표시하지 않고 파일 개수만 보여줍니다.
파일을 드래그앤드롭하거나 클릭해서 선택하세요.

2. 기본 정보

파일에서 추출 가능한 값은 자동 채움, 부족한 값은 수기 입력

보통 계약종료일 다음 날

3. 통상임금/성과상여금 설정

파일에 없거나 귀속기간 판별이 어려운 항목만 별도 입력

성과상여금은 파일에서 자동 반영하지 않습니다. 지급월만으로는 어느 계약기간 근무분인지 판단하기 어렵기 때문입니다. 해당 계약기간 근무분이면서 실제로 이번 퇴직금 산정에 반영해야 하는 금액만 직접 입력하고, 다른 학년도·다른 계약기간 근무분이거나 산정대상기간 밖 지급분이면 0으로 둡니다.
해당 계약기간 근무분만 입력합니다. 이전·이후 근무분이면 0으로 둡니다.
포함 옵션이 켜져 있을 때만 1/12로 반영합니다. 해당 계약기간 근무분만 입력합니다.
월 단위 자료만 있는 경우, 해당 월 n일에 지급된 것으로 간주
정근수당(1/12), 명절휴가비(1/12), 성과상여금 B등급(1/12)을 통상임금에 반영
2026-03-01 이후 신규 임용자의 경우 자동으로 켜지도록 했습니다. 다만 실제 행정 적용 기준에 맞게 직접 변경할 수 있습니다.

4. 항목 분류 검토

파일마다 항목명이 조금씩 달라질 수 있으니 자동 분류를 한 번 확인하세요. 성과상여금은 파일 자동 반영 대상에서 제외됩니다.

파일 분석 후 항목 분류표가 나타납니다.

5. 계산 실행

분류가 맞으면 계산 후 결과 파일을 내려받으세요.

아직 계산되지 않았습니다.

6. 계산 결과

평균임금과 통상임금을 나란히 비교합니다.

계산을 실행하면 결과 요약과 상세 내역이 표시됩니다.
