엑셀 병합행 필터 적용기

엑셀 파일을 올리고, 필터를 걸 헤더 행 번호만 입력하면 해당 시트에 autoFilter 범위를 넣은 파일을 다시 내려받습니다.

현재 파일 없음

1. 파일 올리기

여기에 엑셀 파일을 끌어다 놓으세요
또는 클릭해서 파일을 선택하세요. 지원 형식: .xlsx / .xlsm / .xltx / .xltm
서버 업로드 없음 병합 유지 우선 기존 필터는 덮어쓰기

이 페이지는 워크북을 새로 조립하지 않고, 선택한 시트의 필터 정보만 바꾸는 방식으로 처리합니다. 그래서 병합 셀, 수식, 스타일, 매크로가 있는 파일도 원형을 최대한 유지하는 쪽에 맞춰져 있습니다.

2. 시트와 필터 행 지정

다중 시트 파일이면 적용할 시트를 선택합니다.
예: 헤더가 5행이면 5를 입력합니다.
파일을 먼저 올려 주세요.

3. 동작 방식과 제한

  • 선택한 시트의 실제 사용 범위를 기준으로 시작열 + 입력한 행부터 마지막열 + 마지막행까지 자동필터 범위를 잡습니다.
  • 기존 autoFilter가 이미 있으면 새 범위로 덮어씁니다.
  • 병합 셀은 해제하지 않고 그대로 둡니다.
  • .xls, .xlsb, 비밀번호로 암호화된 파일은 이 페이지에서 바로 처리하지 않습니다.
  • 선택한 헤더 행 자체가 병합되어 있거나, 시트 보호가 걸려 있으면 엑셀에서 필터 사용이 제한될 수 있습니다.
  • 이 HTML은 JSZip 1개를 CDN에서 불러오므로 사용 시 인터넷 연결이 필요합니다.
