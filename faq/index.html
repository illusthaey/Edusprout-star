자주 묻는 질문

필요시 추가됩니다.

이 사이트는 공식 업무 시스템인가요?
아닙니다. 현장 실무를 돕기 위해 개인이 제작한 비공식 참고용 도구입니다. 공식 지침이나 내부 시스템을 대신하지 않으므로 실무자 본인이 항상 지침과 규정 확인을 생활화합시다. 잘못 되도 나는 책임 못 짐.
업로드한 파일은 서버에 저장되나요?
저장되지 않습니다. 서버와 데이터베이스 없이 개발했으며, 업로드 파일은 브라우저 안에서만 처리됩니다.
강원 외 지역에서도 사용 가능한가요?
방과후강사 월보수액 계산기처럼 기준이 전국적으로 동일한 일부 도구는 지역과 무관하게 사용할 수 있지만, 일부 업무는 지역마다 기준·보수표·서식이 다를 수 있습니다. 특히 교육공무직과 관련된 업무 도구는 강원 외 지역에서는 반드시 해당 지역 기준을 직접 확인한 뒤에 사용하시오.
개별 업무 해석·검토 문의도 가능한가요?
지원하지 않습니다. 개별 사례의 판단, 해석, 제출 전 검토, 1:1 업무 교육 지원은 제공하지 않습니다. 오류 제보와 개선 의견은 피드백 가능함.
모바일보다 PC 사용이 권장되나요?
네. 엑셀, PDF, 대량 데이터 정리 기능이 많아 PC 사용을 권장합니다. 모바일에서도 열리지만 작업 안정성과 검토 편의성은 PC가 더 좋습니다.
오류 제보는 어떻게 하나요?
하단의 이메일 주소로 메일 보내주시거나, 우측 상단에 있는 "고주무관에게 사용자 의견 전달하기" 버튼을 누시면 네이버폼 설문조사 응답화면으로 연결됩니다. 네이버 로그인이 불필요하며, 여러 차례 설문 응답이 가능합니다.

기능 문의 시에는 사용한 기능의 링크, 어떤 내용을 입력 혹은 어떤 파일을 업로드한 조건에서 어떤 문제가 발생했는지 설명 바랍니다. 입력 내용 및 파일과 오류 화면 캡처본을 함께 보내 주시는 게 저도 피드백 주기 좋음. 그냥 왜 안 되냐고만 물어보면 저도 모릅니다. 상식적으로 내가 어떻게 알아.
