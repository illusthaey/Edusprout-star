교육공무직 초과근무수당 계산기

날짜와 시간대를 입력하면 법내·법외·야간·휴일근로 시간을 자동 구분하고, 직종·최초임용일 기준 통상임금을 반영해 지급액을 계산합니다.

기본 설정

입력하면 행 날짜 기준으로 근속연수를 자동 계산합니다.
입사일을 비워두면 이 값을 사용합니다.
기본값: 전일제 209 / 1일 5시간 130 / 1일 6시간 156
행별 계산은 각 행 날짜의 스냅샷을 사용합니다.
키보드로 직접 입력할 수 있습니다. 예: 08:40 또는 840
입력한 시간대가 평일 법내·법외 기준 계산에 사용됩니다.
소정근로 실근무시간 계산과 행별 휴게 공제에 함께 사용합니다.
현재 기본설정의 정규근무시간과 휴게시간을 모든 행에 반영합니다.
수기 시급을 사용할 때만 적용됩니다.
· 주말은 자동으로 휴일로 계산합니다. 평일 공휴일·기관 휴무일은 행에서 휴일을 직접 체크하세요.
· 평일의 법내/법외 구분은 기본설정의 정규근무시간에서 휴게를 제외한 1일 실근무시간을 기준으로 주 40시간 한도 안에서 계산합니다.
· 10원 단위 미만은 각 항목별로 절삭한 뒤 합산합니다.
· 자정을 넘기는 근무는 선택한 날짜 기준으로 묶어 계산합니다. 날짜별 엄밀 분리가 필요하면 행을 나눠 입력하세요.
· 행에서는 초과근무 시간만 입력하고, 정규근무시간·휴게시간은 기본설정의 적용값을 사용합니다.
· 기본설정을 바꾼 뒤에는 근무시간·휴게 일괄 적용 버튼을 눌러야 기존 행에도 반영됩니다.
· 운동부지도자는 근속수당 미적용 직종이라 최초임용일 또는 수동 근속연수와 무관하게 근속수당을 0원으로 계산합니다.
